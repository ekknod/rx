# RX 
##### process/library/input handler for linux
----
## building
* git clone https://github.com/ekknod/rx.git
* cd rx
* mkdir build && cd build
* cmake ..
* make
----
## what in future?
* better documentation
* windows support (?)
----
## features
* list processes
* list libraries
* remote process handling ( memory, libraries, exports )
* listen / send input ( mouse / keyboard )
