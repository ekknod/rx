# RX

----
##
##
**miminimal process/library/input handler for linux**

----
## usage
1: git clone https://github.com/ekknod/rx  
2: cd rx  
3: mkdir build && cd build  
4: cmake ..  
5: make  
* example usage ( gcc main.c librx.a )
